Authentication with fastapi and jwt
=======
## Dependencies
* bcrypt==3.1.7  
* databases==0.5.3  
* fastapi==0.68.0  
* Hypercorn==0.11.2  
* jose==1.0.0  
* pydantic==1.8.2  
* python-jose==3.3.0  
* python-multipart==0.0.5  
* SQLAlchemy==1.4.20  
* starlette==0.14.2  
* uvicorn==0.15.0
